The Buttons at the top listed from left to right are as follows:

Pan/Select Node (Scroll to zoom)
Move node
Create node
Create and connect random number of nodes
Connect selected node to another (must have a selected node)
Run/pause simulation
Stop simulation
Export node network to text
Load node network from text
Help


Delay specifies the network delay of that node, the sum of the delay of 2 nodes is the amount of ticks required for a packet to go from one to the other
Power is the hashing power of the node, the higher it is, the quicker it can generate a block
A Tick is a time unit in the simulation, currently 20ms
